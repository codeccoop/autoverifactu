# Verifactu

Verifactu és un módulo de Dolibarr sencillo que permite generar registros de facturación según el sistema [VERI\*FACTU](https://sede.agenciatributaria.gob.es/Sede/iva/sistemas-informaticos-facturacion-verifactu.html) y su envio telemático a la Agencia Tributaria (AEAT), integrado con el sistema de facturación de Dolibarr.
